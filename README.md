# Skillbit ⚡️

A Micro-Learning platform for high-performers. One 5-minute lesson per day. Habit-forming, gamified, and effective.

## Features
- **8 Skill Tracks**: Web Dev, UI/UX, Marketing, Finance, Copywriting, Speaking, AI, Entrepreneurship.
- **80 Real Lessons**: Curated educational content with 300+ words per lesson.
- **XP & Streak System**: Earn XP per lesson, maintain your daily flame.
- **Quiz Engine**: 3 multiple-choice questions per lesson. Must get 2/3 to pass.
- **Supabase Powered**: Ready for authentication and progress persistence.

## Tech Stack
- HTML5 / CSS3 (Electric Indigo & Lemon Lime theme)
- Vanilla JavaScript
- Supabase (Backend/Database)

## Installation & Setup
1. Clone the repository.
2. Create a Supabase project.
3. Run the SQL in `schema.sql` in the Supabase SQL Editor.
4. Update `shared/app.js` with your `SUPABASE_URL` and `SUPABASE_KEY`.
5. Open `landing/index.html` in your browser.
